Mobile Application Management

Workspace ONE UEM REST APIs allows you to manage the end-to-end functionalities of Mobile Application Management (MAM) features. Using these APIs, you can upload internal and public applications, assign, and manage applications on the devices. Application details can be viewed in the following path on the Workspace ONE UEM: As an administrator, you need to enable Apps role permissions in Accounts > Administrators > Roles page to access MAM functionalities through APIs.
